# Convention
1. C++ coding style
 - Large carmel case (but when usign bool var, you should add 'b'prefix)
 - and then follow unrealengine coding style

2. BP naming Style
 - Animation : ABP_
 - BehaviorTree : BT_
 - BlackBoard : BB_
 - BlendSpace : BS_
 - Widget : WBP_
 - Other : BP_
